# Normal Mode

## Move start/end of file

| Atalho | Ação                            | Dica               | Atalho 2            |
| ------ | ------------------------------- | ------------------ | ------------------- |
| `1G`   | Mover para o início do arquivo  | Line **1**, **G**o | `<ctrl>` + `<home>` |
| `G`    | Mover para o final do arquivo   | **G**o             | `<ctrl>` + `<end>`  |

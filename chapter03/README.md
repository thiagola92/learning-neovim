# Normal Mode

## Move across the screen

| Atalho | Ação                          | Dica       |
| ------ | ----------------------------- | ---------- |
| `H`    | Mover para o topo da tela     | **H**igh   |
| `M`    | Mover para o meio da tela     | **M**iddle |
| `L`    | Mover para o inferior da tela | **L**ow    |
